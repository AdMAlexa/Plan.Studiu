6. HTML Styling and Colors


6.1 style attribute


Atributul style este folosit pentru a adauga stiluri unui element, cum ar fi culoarea, fontul, dimensiunea si multe altele.

style extern - defineste reguli de stil intr-un fisier.css separat si apoi includem acel fisier in documentul nostru HTML folosind eticheta HTML "link".
style intern - defineste reguli de stil in secțiunea <head> a documentului HTML folosind atribut style
style inline - defineste reguli de stil impreună cu elementele HTML folosind atributul de style.


exemplu Atributul style



HTML colors


Culorile sunt specificate cu nume de culori sau cu valori RGB, RGBA, HEX, HSL si HSLA.


RGB (red, green, blue). Fiecare parametru (red, green, blue) defineste intensitatea culorii cu o valoare intre 0 si 255.


RGBA(red, green, blue, alpha). Aici avem in plus parametrul alfa care este un numar intre 0,0 (complet transparent) si 1,0 (deloc transparent).


HEX colors notat astfel #rrggbb. Exact ca si la RGB, rr reprezinta red, gg reprezinta green, bb reprezinta blue, acestea sunt valori hexazecimale cuprinse intre 00 si ff


HSL colors (hue, saturation, lightness).
hue - este un grad pe roata de culori de la 0 la 360. 0 este roșu, 120 este verde și 240 este albastru.
saturation - este o valoare procentuală. 0% înseamnă o nuanță de gri, iar 100% este culoarea completă.
lightness - este, de asemenea, o valoare procentuală. 0% este negru, iar 100% este alb.


HSLA (hue, saturation, lightness, alpha). La fel ca si la RGBA si aici avem in plus parametrul alfa care este un numar între 0,0 (complet transparent) si 1,0 (deloc transparent).

15.HTML Encoding


Pentru a afisa corect o pagina html, HTML-ul nostru trebuie sa stie ce set de caractere sa foloseasca iar acest
lucru este specificat in <meta> tag in partea de <head>: <meta charset="UTF-8">


URL encoding este traducerea caracterelor neimprimabile sau caractere cu semnificatie specială in URL-uri într-o reprezentare
fara ambiguitate si acceptata universal de browserele web si servere.


    Aceste caractere includ:

  1. ASCII control characters - Caractere neimprimabile utilizate de obicei pentru controlul iesirii. Interval de caractere
    00-1F hex (0-31 zecimale) și 7F (127 zecimale)
  2. Non-ASCII control characters - Acestea sunt caractere dincolo de setul de caractere ASCII de 128 de caractere.
    Acest interval face parte din setul de caractere ISO-Latin si include intreaga „jumatate superioara” a setului
    ISO-Latin 80-FF hex (128-255 zecimale).
  3. Reserved characters - Acestea sunt caractere speciale, cum ar fi dollar sign, ampersand, plus, common,
    forward slash, colon, semi-colon, equals sign, question mark, and “at” symbol.Toate acestea pot avea semnificații
    diferite în interiorul unei adrese URL, asa ca trebuie să fie encoded
  4. Unsafe characters - Acestea sunt space, quotation marks, less than symbol, greater than symbol, pound character,
    percent character, Left Curly Brace, Right Curly Brace, Pipe, Backslash, Caret, Tilde, Left Square Bracket,
    Right Square Bracket, Grave Accent. Aceste caractere prezinta posibilitatea de a fi intelese gresit în URL-uri
    din diverse motive astfel si acestea trebuie sa fie encoded

URL encoding inlocuieste caracterele ASCII nesigure cu un semn de procent si doua cifre hexazecimale.


URLs nu pot contine spatii prin urmare encoding-ul inlocuieste aceste spatii cu un semn plus.
