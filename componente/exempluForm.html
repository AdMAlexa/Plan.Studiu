Exemplu formular

Elementul <input> este cel mai folosit element de formular si acesta poate
fi afisat in mai multe moduri, in functie de tipul de atribut.

Obsevam ca avem elmentul <label>, acesta defineste elementul formular <input>

Pentru a trimite acest formular catre server avem nevoie de un element <button> impreuna
cu atributul type="submit" sau elementul <input> impreuna cu atributul type="submit" .

Un alt atribut utilizat imputului este reset acesta ajuta userul sa reseteze totul in cazul in care a gresit ceva.
